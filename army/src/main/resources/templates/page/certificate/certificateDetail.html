큐넷 신청 방법

  • 1단계

    Q-net 사이트 (https://www.q-net.or.kr/man001.do?imYn=Y&gSite=Q) 접속

  • 2단계

    Q-net 홈페이지 오른쪽 위 [회원가입] 클릭 회원선택 > 약관동의 > 본인인증 > 신청서작성 > 가입완료 진행

  • 3단계

    Q-net 메인 홈페이지 [원서접수] 클릭

  • 4단계

    본인에게 해당되는 응시할 자격증 시험 [접수하기] 클릭

  • 5단계

    본인이 보고자 하는 시험 명을 [응시 종목명]에 입력
    또는 오른쪽 탭에서 해당 자격증 찾기 클릭

  • 6단계

    학력정보 입력 > 경력정보 입력 > 수상경력 입력 > 진단결과보기 클릭
    # 진단 결과 중 [가능] 내역 선택

  • 7단계

    응시자 기초정보 입력
    응시 종목 및 등급과 면제과목 확인 후 편의제공여부(장애여부 등) 선택
    입력 후 [입력내용] 확인

  • 8단계

    지역선택 > 시/군/구 선택 > 조회 클릭 > 시험장소 확인 > 선택 클릭
    선택 후 [접수하기] 클릭

  • 9단계

    결제 비용 확인 후
    [결제방법](신용카드/실시간 계좌이체/Foreign Card)을 선택 후 결제

  • 10단계

    마이페이지 > [원서접수] 내용 확인!
